# EIF CogAI 2026

Single-page site for the **2nd Exploring Interdisciplinary Frontiers (EIF): Cognitive Science, Computational Modeling, and Artificial Intelligence**, 25–26 June 2026, University of Oxford.

## Local preview

Open `index.html` directly, or serve the folder:

```bash
python -m http.server 5173
```

then visit `http://localhost:5173`.

## Structure

- `index.html` — full site (CSS + JS inline)
- `photos/` — speaker, organiser and venue photographs
- `logos/` — partner-institution logos and the EIF brain mark

## Deploy

This is a fully static site. To deploy via **GitHub Pages**: Settings → Pages → Build from `main` branch (root). No build step required.
